Comment exécuter le programme
1. Exécutez le fichier Server.java
2. Pour chaque joueur, exécutez le fichier Client.java (Pour simuler un jeu en réseau) (En réalité, chaque joueur exécutera le fichier Client.java)
3. Créez un nouveau joueur ou utilisez un joueur existant en saisissant son pseudo
(Pour sélectionner des options, écrivez simplement le numéro de l'option désirée.)
